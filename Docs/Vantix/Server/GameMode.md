# GameMode

`Vantix.Server.GameMode`

Active game mode — determines which spawn pool is used. A future round-manager system selects the mode per match.

## Fields

| Name | Summary |
|------|---------|
| `Competitive` | Round-based CT vs T. Joiners are assigned alternately. |
| `Deathmatch` | Free-for-all — everyone uses the Deathmatch pool. |
