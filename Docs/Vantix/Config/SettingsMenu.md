# SettingsMenu

`Vantix.Config.SettingsMenu`

Settings menu, opened via `ToggleKey` (ESC by default). Code-driven UI; live-applies on change plus an explicit Save. A quality preset sets every individual value; tweaking one switches the preset to Custom.

## Properties

| Name | Summary |
|------|---------|
| `IsAnyOpen` | Set while the menu is open; other systems consult this to block input. |

## Methods

| Name | Summary |
|------|---------|
| `AddDropdown(VBoxContainer, string, String[])` | Adds a labelled dropdown (OptionButton) row and returns the button. |
| `AddSectionHeader(VBoxContainer, string)` | Adds a styled section header (separator + label) into the given VBox. |
| `AddSlider(VBoxContainer, string, float, float, float)` | Adds a labelled horizontal slider row with a value label, returns both. |
| `BuildControlsTab()` | Builds the Controls tab (mouse sensitivity, FOV). |
| `BuildDisplayTab()` | Builds the Display tab. |
| `BuildDynamicLists()` | Builds the resolution + fps-cap dropdowns for `MonitorIndex`. Resolutions come from native enumeration (`EnumModes`/`EnumModes`) for the exact advertised modes in true physical pixels; falls back to a hardcoded candidate list filtered by native size when no native backend exists (macOS/Wayland). |
| `BuildGraphicsTab()` | Builds the Graphics tab. |
| `BuildMonitorLabels()` | Builds Monitor-dropdown labels (Windows "Display N" + physical native + refresh + primary tag). Uses Win32/xrandr for physical pixels and primary detection because Godot's APIs mis-report under per-monitor DPI. |
| `BuildUI()` | Constructs the full menu layout (panel, tabs, save/close row). |
| `FormatResolutionLabel(Vector2I)` | Formats "1920×1080 (16:9)"; drops the tag for unrecognised aspect ratios. |
| `FpsCapToIndex(int)` | Finds the index of the given FPS cap in the dynamic FPS-cap list. |
| `IndexToVSync(int)` | Maps a dropdown index back to a Godot VSyncMode. |
| `IndexToWindowMode(int)` | Maps a dropdown index back to a Godot WindowMode. |
| `MarkCustomPreset()` | A single control was tweaked → preset switches to Custom. |
| `NewTabPage(string)` | Creates a tab page (ScrollContainer + VBox). The node name becomes the tab title. |
| `OnAaChanged(long)` | Anti-aliasing mode changed. |
| `OnAdsDofChanged(long)` | ADS depth-of-field toggle changed. |
| `OnAdsFovZoomChanged(long)` | ADS FOV zoom toggle changed. |
| `OnAoChanged(long)` | Ambient occlusion toggle changed. |
| `OnAutoExposureChanged(long)` | Auto-exposure toggle changed. |
| `OnBrightnessChanged(double)` | Brightness slider changed. |
| `OnCameraShakeChanged(long)` | Camera shake toggle changed. |
| `OnChromaticAberrationChanged(long)` | Chromatic aberration toggle changed. |
| `OnCloudShadowDistChanged(long)` | Cloud-shadow distance preset changed (Near/Medium/Far). |
| `OnCloudShadowsChanged(long)` | Cloud shadows toggle changed. |
| `OnDirectionLeanChanged(long)` | Direction lean toggle changed. |
| `OnDisconnectPressed()` | Disconnect button: closes the menu, then asks NetMain to tear down the connection. |
| `OnDustMotesChanged(long)` | Dust motes toggle changed. |
| `OnFilmGrainChanged(long)` | Film grain toggle changed. |
| `OnFovChanged(double)` | FOV slider changed. |
| `OnFpsCapChanged(long)` | FPS cap dropdown changed; defers apply while menu is open (MenuFpsCap stays in effect). |
| `OnGodRaysChanged(long)` | God rays toggle changed. |
| `OnHudMarginHChanged(double)` | HUD horizontal margin slider changed (HudCs2 reads per frame → applies immediately). |
| `OnHudMarginVChanged(double)` | HUD vertical margin slider changed. |
| `OnLensFlareChanged(long)` | Lens flare toggle changed. |
| `OnMenuFpsCapChanged(long)` | Menu FPS cap dropdown changed; takes effect immediately if the menu is open. |
| `OnMonitorChanged(long)` | Monitor dropdown changed: updates the active monitor, rebuilds the resolution list, picks the closest valid entry, and applies so the window jumps to the new screen. |
| `OnMotionBlurChanged(long)` | Motion blur toggle changed. |
| `OnMouseInertiaChanged(long)` | Mouse inertia toggle changed. |
| `OnPostProcessingChanged(long)` | Post-processing master toggle changed — disables the whole `PostProcessEffect` dispatch when Off. |
| `OnPresetChanged(long)` | Preset dropdown changed: applies preset values, refreshes controls and re-applies settings. |
| `OnReflectionProbesChanged(long)` | Reflection-probe atlas quality changed; takes effect on next level reload. |
| `OnReflectionsChanged(long)` | Reflections toggle changed. |
| `OnRenderScaleChanged(double)` | Render-scale slider changed: updates value label and applies. |
| `OnResolutionChanged(long)` | Resolution dropdown changed. |
| `OnSavePressed()` | Save button: persists settings and flashes a "Saved" indicator for 2s. |
| `OnSensChanged(double)` | Sensitivity slider changed. |
| `OnShadowsChanged(long)` | Shadow quality changed. |
| `OnSharpeningChanged(long)` | Sharpening toggle changed (also gated off under an FSR upscaler in Settings.Apply). |
| `OnShowDebugBarChanged(long)` | Debug bar visibility toggle changed. |
| `OnShowNetGraphChanged(long)` | Net graph visibility toggle changed. |
| `OnSkyChanged(long)` | Sky toggle changed. |
| `OnSprintSwayChanged(long)` | Sprint sway toggle changed. |
| `OnTeamGlowChanged(long)` | Team-glow toggle changed (PuppetPlayer reads Settings.TeamGlow per update; no Apply needed). |
| `OnUiMsaaChanged(long)` | UI MSAA dropdown changed (drives root Viewport.Msaa2D). |
| `OnUiScaleChanged(double)` | UI-scale slider changed (drives Window.ContentScaleFactor). |
| `OnUpscalerChanged(long)` | Upscaler (Bilinear/FSR1/FSR2) changed. |
| `OnVSyncChanged(long)` | VSync mode dropdown changed. |
| `OnViewBobChanged(long)` | View bob toggle changed. |
| `OnViewmodelScaleChanged(double)` | Viewmodel-scale slider changed (weapon viewport scales independently from the world). |
| `OnVignetteChanged(long)` | Vignette toggle changed. |
| `OnVolumetricFogChanged(long)` | Volumetric fog quality changed. UI indices 0..3 map directly to enum Off/Low/Medium/High. |
| `OnWindowModeChanged(long)` | Window mode dropdown changed. |
| `Open()` | Opens the menu programmatically (used by the main menu Settings button). |
| `PullStateFromSettings()` | Seeds every UI control with the current Settings values without firing change events. |
| `RefreshGraphicsControls()` | Sets all graphics controls to the current Settings values (including preset). |
| `ResolutionToIndex(Vector2I)` | Finds the index of the given resolution in the dynamic resolutions list. |
| `RestoreGodotObjectData(Bridge.GodotSerializationInfo)` | — |
| `SaveGodotObjectData(Bridge.GodotSerializationInfo)` | — |
| `SetOpen(bool)` | Opens/closes the menu, managing mouse mode (snapshot+restore) and the menu/game FPS-cap swap. |
| `VSyncToIndex(DisplayServer.VSyncMode)` | Maps a Godot VSyncMode to a dropdown index. |
| `WindowModeToIndex(DisplayServer.WindowMode)` | Maps a Godot WindowMode to a dropdown index (Windowed/Fullscreen/Exclusive). |
| `_Ready()` | Builds the menu, hides it, and seeds the controls from current Settings. |
| `_UnhandledInput(InputEvent)` | Handles the toggle key to open/close the menu. |
