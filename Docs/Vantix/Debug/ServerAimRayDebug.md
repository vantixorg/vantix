# ServerAimRayDebug

`Vantix.Debug.ServerAimRayDebug`

Draws a yellow ray (ImmediateMesh) from the server eye in the server's believed aim direction, from `LastSelfSnap` (eye = Snap.Pos + StandEyeHeight). Active only when `sv_debug_aim_ray` is set.
