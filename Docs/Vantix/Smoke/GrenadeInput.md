# GrenadeInput

`Vantix.Smoke.GrenadeInput`

Per-tick grenade input (slot active, throw held, dt) for the charge logic.

## Fields

| Name | Summary |
|------|---------|
| `Dt` | Tick delta time in seconds. |
| `SlotActive` | True while the grenade slot is selected. |
| `ThrowHeld` | True while the throw button is held. |
