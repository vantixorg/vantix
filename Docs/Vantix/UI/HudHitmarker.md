# HudHitmarker

`Vantix.UI.HudHitmarker`

Center-right hitmarker showing "-X HP" when the local player lands a hit; headshot gold, body white, stacked oldest-to-newest, fades after `HoldSec`. Driven by `OnHit`, which the server sends only to shooter and victim.
