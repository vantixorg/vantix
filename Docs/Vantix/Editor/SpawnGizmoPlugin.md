# SpawnGizmoPlugin

`Vantix.Editor.SpawnGizmoPlugin`

Draws a wireframe outline of `Size` for every `Spawn` node. Registered by `SpotsGizmoPlugin`. Green so spawn regions read distinctly from BombSpot (red) and Zone (cyan) in the editor viewport.
